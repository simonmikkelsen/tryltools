Command line utility written in Python, that takes a number of lines
on stdin, puts them in random order and prints them.

The utility is very simple, and will hold everythin in memory.
This gives good randomness, but don't feed it more data than
you have memory.

GPL3 by Simon Mikkelsen http://mikkelsen.tv/simon/
Currently hosted on http://code.google.com/p/tryltools/
