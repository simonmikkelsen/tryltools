The library string magic, is made for the mmv command, which can
mass rename files. It can however be used on any string you like.

It is written in Python and GPL3 by Simon Mikkelsen
http://mikkelsen.tv/simon and is currently hosted on
http://code.google.com/p/tryltools/

The status of the code is alpha.

Please send bugreports, feature suggestions and most of all code :-)
